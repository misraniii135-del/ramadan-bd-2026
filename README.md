# রমজান ২০২৬ - ডিজিটাল প্রিমিয়াম (Ramadan 2026 Premium)

একটি আধুনিক এবং প্রিমিয়াম ইসলামিক রমজান অ্যাপ্লিকেশন যা রমজান ২০২৬ এর জন্য বিশেষ ভাবে তৈরি। এটি ব্যবহারকারীদের সেহরি, ইফতারের সময়সূচী প্রদান করার পাশাপাশি AI সহকারীর মাধ্যমে ইসলামিক বিভিন্ন প্রশ্নের উত্তর দিতে সক্ষম।

## বৈশিষ্ট্যসমূহ (Features)
- **নির্ভুল সময়সূচী:** বাংলাদেশের ৬৪ জেলার সেহরি ও ইফতারের নির্ভুল সময়সূচী।
- **ইসলামিক সহকারী:** Gemini AI চালিত স্মার্ট সহকারী যা আপনার ইসলামিক প্রশ্নের উত্তর দেবে।
- **অনুদান সিস্টেম:** সরাসরি অ্যাপ থেকে বিভিন্ন ইসলামিক প্রোজেক্টে অনুদান দেওয়ার এবং ডিজিটাল স্বীকৃতি কার্ড পাওয়ার সুযোগ।
- **যাকাত ক্যালকুলেটর:** আপনার সম্পদের সঠিক যাকাত দ্রুত হিসাব করার সহজ টুল।
- **কার্ড জেনারেটর:** যেকোনো দিনের সময়সূচী বা আপনার অনুদান কার্ড হিসেবে ডাউনলোড ও শেয়ার করার সুবিধা।

## টেকনোলজি স্ট্যাক (Tech Stack)
- **Frontend:** React 19, Tailwind CSS
- **AI Integration:** Google Gemini SDK (@google/genai)
- **Deployment:** Netlify (CI/CD support)
- **Build Tool:** Vite

## কিভাবে সেটাআপ করবেন (Setup Instructions)
১. এই রিপোজিটরি ক্লোন করুন।
২. টার্মিনালে `npm install` কমান্ড চালিয়ে ডিপেন্ডেন্সি ইনস্টল করুন।
৩. স্থানীয়ভাবে চালাতে `npm run dev` কমান্ড ব্যবহার করুন।
৪. প্রোডাকশনে হোস্ট করার সময় Netlify বা অন্য প্ল্যাটফর্মে `API_KEY` এনভায়রনমেন্ট ভেরিয়েবলটি সেট করুন।
